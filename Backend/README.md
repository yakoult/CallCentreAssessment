## Requirements

- .NET 7 SDK
- Microsoft SQL Server

## Getting Started

Ensure dependencies are installed with:

```bash
dotnet restore
```

Run the development server with:

```bash
dotnet run --project ./Assessment.Api
```

### Stack

- .NET and EF 7.0
- MediatR
- Serilog, OpenTelemetry, with Jaeger integration
- Swagger
